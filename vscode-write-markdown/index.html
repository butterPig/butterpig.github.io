使用vscode以及idea的markdown编辑插件，效果不错还能实时预览

butterPig 发表于 2021-02-05 本文总阅读量次

因为第一次写markdown，在网上各种百度，想看看到底哪些专门的markdown编辑器好用，下载了好几种。但是真正的使用体验完全没有文章里说的那些惊艳激动人心。最终想回归到平时使用最多的开发环境IDE中寻找答案。

在vscode里面安装markdown支持插件也非常容易。直接在vscode的扩展里面搜索markdown，选着最上面下载次数最多的那个Markdown All in One就全部搞定，这名字有点全家桶的味道。用起来效果不错，在右边就可以实时的预览，没有延迟。左边打字右边马上就能渲染出来，没有网络的时候也可以使用。据说有很多markdown实时预览是需要有网络的。

另外，因为平时开发使用idea比较多，就把markdown用idea打开试了下，idea提示自动下载插件，然后也和vscode一样，比较稳定方便的书写和预览markdown了。感觉不错。

我还安装了另几个书写markdown的编辑器，反而效果没有太好。比如markdownPad，安装好一进去就疯狂报错，让跳转到指定页面解决问题，然而啥也解决不了。官方说是因为win10系统的原因导致的，我用win7兼容模式打开也不停的提示渲染错误。

还安装了印象笔记，网上宣传的不错，不过安装之后要注册登录才行，然后免费版会提示你要升级plus版，还会弹出广告框之类的，实在忍不了卸载了。

听说notepad++里面也有markdown插件，下载安装之后，发现notepad里面的markdown插件虽然能预览，但是不能实时的渲染，每次想看最新渲染，必须把预览窗关了再重新打开。。。总之对于一个markdown新手来说，感觉vscode里面的免费markdown编辑器已经挺满意的了。
